# Dataset

本プロジェクトでは公開画像データセットを利用。

使用データセット：
- Weather Image Recognition(kaggle)  
https://www.kaggle.com/datasets/jehanbhathena/weather-dataset?utm_source=chatgpt.com&select=dataset  
使用クラス：snow, rain

- Weather Dataset for Image Classification(kaggle)  
https://www.kaggle.com/datasets/orvile/weather-dataset-for-image-classification/data  
使用クラス：cloudy, shine, rain
*shineクラスはsunnyとして利用。


容量の都合上、画像本体はGitHubには含めていません。
